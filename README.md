# C++ Sandbox

Repo with poc and tests about C++ 

### Usage

* Create file
```
$file.cpp
```

* Compile
```shell
g++ $file.cpp
```

* Compile and Name the output
```shell
g++ -o $output  $file.cpp
```

* Execute code
```shell
./a.out
```
